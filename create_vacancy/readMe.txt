== Описание ==

Плагин "Добавление Вакансий" позволяет вам легко создавать и управлять вакансиями на вашем сайте WordPress. Вы можете использовать этот плагин для добавления, редактирования и отображения вакансий.

= Особенности =

- Создание пользовательского типа записи для вакансий.
- Административная страница для добавления и управления вакансиями.
- Удобный интерфейс для добавления названия и описания вакансии.
- Отображение вакансий на вашем сайте.

== Установка ==

1. Загрузите папку `add-vacancy` в директорию `/wp-content/plugins/`.
2. Активируйте плагин в разделе 'Плагины' в WordPress.
3. Создайте новую страницу и используйте шорткод `[add_vacancy_form]` для отображения формы вакансии.

== Использование ==

1. После активации перейдите в меню "Плагин Вакансий" в административной части WordPress.
2. Используйте форму "Добавить Вакансию" для добавления новых вакансий.
3. Опубликованные вакансии будут отображаться на вашем сайте.

= Как добавить новую вакансию? =

1. Перейдите в меню "Плагин Вакансий" в административной части WordPress.
2. Используйте форму "Добавить Вакансию" для заполнения данных о вакансии и ее публикации.

= Как отобразить вакансии на сайте? =

Для отображения формы вакансии на странице, используйте шорткод `[add_vacancy_form]`.
